# 星辰恋曲的白色永恒-雪雪mod
本模组是以《星辰恋曲的白色永恒》为参考设计的模组，以主角“雪雪”为操控角色游玩杀戮尖塔2，并根据galgame内的剧情设定了独特的体系和能力，希望你能玩得开心。

目前模组测试的测试可能不够完全，所以在游玩过程中有任何的bug或者疑惑还请随时前往bilibili视频和私信或在此提出，十分感谢。

本模组中使用的一切图片均来源于galgame“星辰恋曲的白色永恒”,少数ai生成以及手绘。

模组中涉及动画等的部分代码使用ai生成。 Parts of the code related to animation for this mod were generated or optimized with the help of AI (e.g., Gemini).
